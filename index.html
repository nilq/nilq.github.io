Evolutionary Neural Networks

As a new and improved iteration of a simple biology project I had made around 2016(7th grade), I have been experimenting with various simulated environments hosting artificial life forms. The artificial 'life' are based on damp-weighted recurrent neural networks (DWRAON networks for short) that have been hooked up to a virtual nerve-system and some simple legs for the creatures. This simulation has reached the state where it pretty much fulfills the purpose of a cozy aquarium.

The Wu Language

In 2017 I ignited my passion for programming language design. Around that time I had attended a lot of game jams, and liked developing small video games for game jam competitions and as a hobby. I developed Wu using the early versions of the Rust language, and designed it to be what I saw as the perfect language for my game development and AI-programming needs. The language transpiles to the Lua programming language, which is used for various game engines and machine learning tools like Torch. After the first stable version of the language it has gotten a considerable amount of attention on GitHub in the form of stars from programming language and game development enthusiasts and professionals.
